========================================
使用 MongoDB Atlas 云数据库（最简单）
========================================

步骤 1: 注册账号（免费）
访问: https://www.mongodb.com/cloud/atlas/register
点击 "Try Free" 注册账号

步骤 2: 创建免费集群
1. 选择 "Build a Database"
2. 选择 "M0 FREE" 免费套餐
3. 选择云服务商和区域（可以选择最近的，如 AWS 的 us-east-1）
4. 点击 "Create"

步骤 3: 创建数据库用户
1. 用户名: admin (或任意)
2. 密码: 设置一个密码（记住它！）
3. 点击 "Create Database User"

步骤 4: 配置网络访问
1. 点击 "Network Access"
2. 点击 "Add IP Address"
3. 选择 "Allow Access from Anywhere" (输入 0.0.0.0/0)
4. 点击 "Confirm"

步骤 5: 获取连接字符串
1. 点击 "Database" 回到数据库页面
2. 点击 "Connect"
3. 选择 "Connect your application"
4. 复制连接字符串，类似：
   mongodb+srv://admin:<password>@cluster0.xxxxx.mongodb.net/?retryWrites=true&w=majority

步骤 6: 配置应用
1. 将连接字符串中的 <password> 替换为你设置的密码
2. 在连接字符串末尾添加数据库名称，例如：
   mongodb+srv://admin:你的密码@cluster0.xxxxx.mongodb.net/story-make?retryWrites=true&w=majority
3. 在项目根目录创建 .env 文件
4. 添加以下内容：
   MONGODB_URI=mongodb+srv://admin:你的密码@cluster0.xxxxx.mongodb.net/story-make?retryWrites=true&w=majority

步骤 7: 重启服务器
运行以下命令重启后端服务器：
  pkill -f "node server.js"
  npm start

完成！现在刷新浏览器页面即可使用。
